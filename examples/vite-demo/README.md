# Table Craft Demo

Minimal Vite + React + Tailwind app to develop and test `react-table-craft` locally.

## Dev workflow

**Terminal 1** — Watch and rebuild the library on changes:

```bash
cd ../.. && npm run dev
```

**Terminal 2** — Run the demo app:

```bash
npm run dev
```

Open http://localhost:5173/

## First-time setup

1. Build the library: `cd ../.. && npm run build`
2. Install demo deps: `npm install`
